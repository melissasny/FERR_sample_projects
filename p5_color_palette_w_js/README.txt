This is an open-ended assignment, so feel free to be creative.  Here's a completed example:

https://googledrive.com/host/0B7yK_e1LsbdVMjZGeG5ST1BMNWM/index.html

Here are some suggestions (in order of difficulty):

1. Make an text field where you can enter a color name, then pressing ENTER sets the background to that color.
2. Make a color palette builder (where you can enter multiple colors, one at a time).
3. Make a color palette builder (where you can enter multiple colors all at once).
4. Make it so you can click on a color to select it, then press DELETE (or BACKSPACE) to remove it.
5. Make it so you can click on a color to select it, then press arrows to rearrange them.
